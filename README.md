# README

## Instructions to install:

$ ./getmenode.sh
$ npm install all

## Instructions to execute:

$ ./runmenode.sh

// alternative for home pc:
$ npm start 

## To test:

$ npm test
-- these tests should always pass (when the server is up and running ofc)
